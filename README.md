## Installation

Clone the repository:

```bash
git clone https://github.com/Zannatmawa/cemzo-product-catalog
```

Go to the project folder:

```bash
cd repository-name
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
